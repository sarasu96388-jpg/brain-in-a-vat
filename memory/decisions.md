## 2026-04-01

| 决策 | 理由 |
|------|------|
| **废弃棋盘种植系统，改为方案 B：植物通过建造系统完成居民订单后获取** | 1. 棋盘种植导致注意力分散，玩家踩种植格时无法明确感知是亏是赚；2. 植物关卡有限导致溢出与短缺并存；3. 新手认知负担过重；4. 种植作为买量副玩法不应与核心循环平级，会模糊 MGO 核心循环焦点 |
| **核心循环保持清晰：掷骰子→踩格→拿钱→建造→完成居民订单→获得植物→战斗/剧情** | 植物是建造的副产品，不是并行循环，玩家注意力始终在棋盘上 |
| **对战偷菜系统需要重新设计** | 原设计依赖"地里种植的作物"，种植废弃后需要改为不依赖田地的对战形式 |
| **植物获取保留一定随机性** | 完成订单后随机获得 1-N 种植物，概率出变异，保留博彩感 |
